# PWA Icons Required

Please add the following icon files to the `/public` directory:

1. `icon-192x192.png` - 192x192px PNG icon
2. `icon-512x512.png` - 512x512px PNG icon
3. `favicon.ico` - Standard favicon

You can generate these from your logo using tools like:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

## Recommended Design

- Simple, clear icon that represents Korii Matcha
- High contrast for visibility on different backgrounds
- Include branding elements (logo, colors)
